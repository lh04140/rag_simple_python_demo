# RAG Simple Python Demo

一个简单的RAG（检索增强生成）技术演示项目，包含PDF生成功能。

## 项目概述

这个项目演示了RAG技术的基本概念，并提供了一个生成RAG技术概述PDF的简单脚本。

## 功能

- 使用ReportLab库生成PDF文档
- 生成RAG技术概述的PDF文档
- 简单的Python脚本，易于理解和扩展

## 文件结构

```
rag_simple_demo/
├── README.md          # 项目说明文档
├── generate_pdf.py    # 生成PDF的主脚本
└── .gitignore         # Git忽略文件配置
```

## 使用方法

### 1. 安装依赖

首先安装所需的Python包：

```bash
pip install reportlab
```

### 2. 运行脚本

运行生成PDF的脚本：

```bash
python generate_pdf.py
```

运行后会在当前目录生成一个名为 `test.pdf` 的PDF文件。

### 3. 查看结果

打开生成的 `test.pdf` 文件，你将看到包含以下内容的RAG技术概述：

- RAG技术的基本概念
- 主要流程（文档加载、文本分块、向量化与存储、检索、生成）
- 应用场景
- LangChain框架介绍

## RAG技术简介

RAG（Retrieval-Augmented Generation，检索增强生成）是一种将信息检索与大语言模型（LLM）结合的技术框架。它通过从外部知识库中检索相关文档，作为生成答案的上下文，有效解决了LLM的幻觉问题，并能够利用私有或实时数据。

### 主要流程

1. **文档加载**：从各种数据源（如PDF、网页、数据库）加载原始文档
2. **文本分块**：将长文档切分为语义完整的文本块
3. **向量化与存储**：使用嵌入模型将文本块转换为向量，存入向量数据库
4. **检索**：用户提问时，将问题向量化，在数据库中检索最相似的文本块
5. **生成**：将检索到的文本块与问题组合成提示词，送入LLM生成答案

### 应用场景

- **智能客服**：基于产品手册回答用户问题
- **企业知识库**：快速检索内部文档，辅助员工决策
- **学术研究**：从论文库中提取信息，辅助写作

## 技术栈

- **ReportLab**：用于生成PDF文档
- **Git**：版本控制
